 🛒 Online Shopping Database Project Using MySQL

 📌 Project Description
This project is a MySQL-based Online Shopping Database Management System. It stores and manages customer, product, order, payment, shipment, and review information. The database is designed using normalization and relationships to ensure data consistency and efficient query execution.

 🎯 Objectives
- Manage customer information
- Store product details
- Process customer orders
- Track payments
- Track shipments
- Maintain customer reviews
- Manage wishlist items

 🛠 Technologies Used
- MySQL 8.0
- MySQL Workbench
- SQL (DDL, DML, DCL)
- Oracle

 📂 Database Tables

1. User
2. Product
3. Category
4. Orders
5. OrderItems
6. Payments
7. Shipments
8. Reviews
9. Wishlist
10. Admin (Optional)

 🔑 Features
- Customer Registration
- Product Management
- Shopping Cart
- Order Management
- Payment Tracking
- Shipment Tracking
- Product Reviews
- Wishlist Management
